## ⚡ About

**Budget App** — это минималистичное и мощное приложение для управления личными финансами.

Ты можешь:
- 📥 добавлять доходы
- 📤 отслеживать расходы
- 📊 видеть аналитику
- 💰 контролировать бюджет

---

## 🎯 Features

- 🔹 Простое добавление транзакций
- 🔹 Категории доходов и расходов
- 🔹 Баланс в реальном времени
- 🔹 Визуальная аналитика
- 🔹 Локальное хранение данных (без сервера)
